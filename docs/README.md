## What The Handoff Docs Are For

The handoff docs explain what was inspected, what changed, and what the next person should know.

They are useful for:

- understanding the codebase quickly;
- seeing current known limits;
- reviewing earlier verification notes;
- checking why some safety gates exist.

## How To Read These Docs

Start with the project overview if you are new.

Then read the codebase scan if you need to work on backend or frontend behavior.

Read the data/citation inventory if your task touches evidence, citations, readiness, or source boundaries.
